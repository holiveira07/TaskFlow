# TaskFlow

## Sobre o projeto

TaskFlow é um sistema de gerenciamento de tarefas desenvolvido como um projeto real durante minha formação em Desenvolvimento de Software.

Este projeto não foi criado apenas para estudar programação. Seu objetivo é documentar todas as etapas de um software profissional, desde o levantamento de requisitos até o deploy da aplicação.

Todo o desenvolvimento será registrado, incluindo:

- Requisitos
- Regras de negócio
- Modelagem de banco de dados
- Arquitetura
- Backend
- Frontend
- APIs
- Decisões técnicas
- Documentação
- Histórico de evolução

---

## Objetivo

Construir um sistema completo utilizando boas práticas de desenvolvimento de software enquanto evoluo como desenvolvedor.
